## Final Deployment

This folder is ready for GitHub Pages deployment.

Included files:
- `index.html`
- `app.js`
- `data.js`
- `.nojekyll`
- `Welfare Simulation Macro Datapack USA 1929-2025.xlsx`

Recommended deployment steps:
1. Create a GitHub repository for the site.
2. Upload all files from this folder to the repository root.
3. In GitHub, open `Settings` -> `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select branch `main` and folder `/ (root)`.
6. Save and wait for publication.

The website link used in the thesis is:
`https://leonardopaolorecro.github.io/WelfareSimulation/`
