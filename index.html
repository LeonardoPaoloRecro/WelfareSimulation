METHOD

How I build the welfare measure, and why I do not rely on GDP alone.

I start from a simple point. GDP tells me how much the economy produces. It does not tell me, by itself, how much of that production ends up as private consumption, how many workers are left out of the labor market, or whether the gains are broadly shared. That is why I use GDP as an output measure, but not as a welfare measure on its own.

The formulas below are written in a way that a non-specialist can still follow. I keep the notation explicit, I explain where each term comes from, and I separate the simpler core welfare version from the fuller version that also adjusts for inequality.

LF_t = L_t / (1 - u_t)
W_t^core = ln(C_t / L_t) + ln(1 - u_t)
W_t = W_t^core + η ln(1 - I_t)

1. What each variable means

Y_t is real GDP, so it measures total production. C_t is real personal consumption, so it is closer to what households privately enjoy. L_t is employment. u_t is the unemployment rate. I_t is the Gini index on a 0 to 1 scale, which I use as a practical inequality proxy.

The reason I keep both employment and unemployment is that they do different jobs in the formula. Employment tells me how thinly private consumption is spread across people in work. Unemployment tells me how much of the labor force is excluded from work altogether.

2. Why I derive the labor force first

If unemployment is a share of the labor force, and employment is the employed part of that labor force, then the accounting identity is:

L_t = (1 - u_t) LF_t

Rearranging gives:

LF_t = L_t / (1 - u_t)

This matters because it lets me move from a measure that only looks at employed workers to a measure that also penalizes labor-market exclusion.

3. Worker-relevant consumption

I do not use consumption per person as the main object, because my question is about worker welfare. Instead I use consumption per labor-force participant:

c_t^worker = C_t / LF_t = C_t (1 - u_t) / L_t

This keeps the worker focus, but it does not ignore unemployment. If unemployment rises, the measure falls even if consumption among the employed looks stable.

4. Core welfare and full welfare

The simpler version is the core welfare measure:

W_t^core = ln(C_t / L_t) + ln(1 - u_t)

The first term captures private consumption per employed worker. The second term captures labor-market access.

The fuller version adds inequality:

W_t = ln(C_t / L_t) + ln(1 - u_t) + η ln(1 - I_t)

Here η is the degree of inequality aversion. I use η = 1 as the baseline. If inequality rises, the final term falls, so welfare falls as well.

5. Why the log form is useful

I use logs because the interpretation of changes becomes cleaner. A change in the welfare index can be split into a consumption part, an employment-access part, and an inequality part:

ΔW_t = Δln(C_t / L_t) + Δln(1 - u_t) + ηΔln(1 - I_t)

This makes the explanation much easier. If welfare changes, I can say whether it moved because private worker consumption changed, because unemployment changed, or because distribution changed.

6. Output multiplier versus welfare multiplier

In a standard Keynesian setting I still care about the output multiplier, because I want to know how strongly government spending moves production:

M_Y = (Y_t - Y_{t-1}) / (G_t - G_{t-1})

But I also want a welfare analogue. First I scale the fiscal change by baseline GDP:

Shock_t = (G_t - G_{t-1}) / Y_{t-1}

Then I define:

M_W^core = (W_t^core - W_{t-1}^core) / Shock_t
M_W = (W_t - W_{t-1}) / Shock_t

This lets me compare a production response and a welfare response inside the same fiscal episode.

7. How I read the result

If the output multiplier is large but the welfare multiplier is small, then fiscal expansion is raising production more than it is raising worker welfare. That is the basic wartime story in the U.S. data, especially in the onset of World War II.

If both multipliers are large, the expansion looks more broad-based. If output rises but welfare falls, that means the composition of growth or the distribution of gains is working against household welfare.

8. Limits I state openly

I do not claim that this formula captures every dimension of welfare. It does not directly include health, mortality, leisure, environmental quality, or wartime fear and destruction. I use the Gini as a practical proxy because a long annual series for consumption inequality is not available in the same way.

So the claim is not that this is the one true welfare function. The claim is that it is a much better measure than GDP alone when the question is whether output growth actually improves worker-centered welfare.
