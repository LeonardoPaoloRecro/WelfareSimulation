Simulation

This section is intentionally manual. You type the values yourself for two periods and the page computes the fiscal shock share, the output multiplier, the core welfare level, the distribution-adjusted welfare level, and the direct welfare responses dW/dG that sit at the center of the thesis.

The idea is simple: this section lets you run a two-year fiscal comparison and inspect how a direct output response differs from a direct welfare response. The tool uses exactly the same algebra shown on the Method page and discussed in the thesis benchmark sections.

You can enter historical values from the U.S. dataset or hypothetical scenario values of your own. The output is therefore a transparent two-period calculation, not an econometric estimate or a realism check. Short limitation: the full welfare measure needs a Gini value in both years. If Gini is missing, the tool still computes the core welfare version.

METHOD

Method.

I start from a simple point. Macroeconomics has spent a great deal of time asking how much output responds to government spending, that is, how large dY/dG is. My question is narrower and more empirical: once a welfare measure W is defined, what does dW/dG look like in the historical U.S. data?

The formulas below are written in a way that a non-specialist can still follow. I keep the notation explicit, I explain where each term comes from, and I separate the direct welfare response from a secondary normalized version that is useful only for readability. The Simulation page applies the same equations to any two-period pair of values the user enters.

MY = ΔY / ΔG
RWcore = ΔWtcore / ΔG
RW = ΔWt / ΔG
ln(Ct / Lt) + ln(1 - ut) + ln(1 - It)

The full welfare measure has three moving parts: private worker consumption, labor-market access, and distribution. The point is simple: output on its own is not enough if consumption is weak, work is not broadly available, or the gains are too concentrated.

1. What each variable means

Yt is real GDP, so it measures total production. Ct is real personal consumption, so it is closer to what households privately enjoy. Lt is employment. ut is the unemployment rate, which enters the welfare formula on a zero-to-one basis even when the charts display it as a percentage. It is the Gini index used in the welfare formula on a zero-to-one basis, although the dataset reports the underlying series on a 0 to 100 scale.

The reason I keep both employment and unemployment is that they do different jobs in the formula. Employment tells me how thinly private consumption is spread across people in work. Unemployment tells me how much of the labor force is excluded from work altogether.

2. Why employment and unemployment both appear

I keep both employment and unemployment because they do different jobs in the welfare design. Employment enters the first term by scaling private consumption to the number of employed workers. Unemployment enters separately because access to work is itself part of welfare.

If unemployment rises, welfare should fall even if average consumption among those still employed does not immediately collapse. That is why the measure remains worker-centered rather than output-centered.

3. Why the first term stays worker-centered

I do not treat GDP as a welfare measure. The first term focuses on private consumption per employed worker, so the measure stays tied to the private material side of the economy rather than to total production alone.

If wartime output rises while private consumption does not, this part of the measure makes that gap visible.

4. Core welfare and full welfare

The simpler version is the core welfare measure:

Wtcore = ln(Ct / Lt) + ln(1 - ut)

The first term captures private consumption per employed worker. The second term captures labor-market access.

The fuller version adds inequality:

Wt = ln(Ct / Lt) + ln(1 - ut) + ln(1 - It)

The inequality adjustment is included directly in the full welfare measure. If inequality rises, the final term falls, so welfare falls as well.

In practice, because the Gini series is reported on a 0 to 100 scale in the dataset, the implemented term is ln(1 - Ginit / 100).

5. Why the log form is useful

I use logs because the interpretation of changes becomes cleaner. A change in the welfare index can be split into a consumption part, an employment-access part, and an inequality part:

ΔWt = Δln(Ct / Lt) + Δln(1 - ut) + Δln(1 - It)

This makes the explanation much easier. If welfare changes, I can say whether it moved because private worker consumption changed, because unemployment changed, or because distribution changed.

The inequality part is only available where the Gini series exists, which is why the thesis keeps the core and full welfare measures distinct.

6. dY/dG versus dW/dG

In the standard Keynesian discussion I still care about the output response, because I want to know how strongly government spending moves production over the selected benchmark window:

MY = ΔY / ΔG

But the central question of this project is the welfare analogue:

RWcore = ΔWtcore / ΔG
RW = ΔWt / ΔG

This lets me compare a production response and a welfare response inside the same fiscal episode using the same change in government spending in the denominator.

In the thesis, these are benchmark episode measures over selected war-onset windows rather than structural econometric multiplier estimates.

For readability, I also keep a secondary scaled version:

Shockt = ΔG / Yt-1
ṘWcore = ΔWtcore / Shockt
ṘW = ΔWt / Shockt

Here, Yt-1 denotes real GDP in the initial year of the benchmark window, so that the fiscal shock is scaled by the size of the economy at the beginning of the episode.

I use that scaled version only as a supplementary comparison tool. The primary benchmark in the thesis is the direct contrast between dY/dG and dW/dG.

7. How I read the result

If dY/dG is large but dW/dG is very small, then fiscal expansion is raising production much more than it is raising worker welfare. That is the basic wartime story in the U.S. data, especially in the onset of World War II.

If both responses are strong, the expansion looks more broad-based. If output rises but welfare barely moves, or even falls, that means the composition of growth or the distribution of gains is working against household welfare.

8. Limits I state openly

I do not claim that this formula captures every dimension of welfare. It does not directly include health, mortality, leisure, environmental quality, or wartime fear and destruction. I use the Gini as a practical proxy because a long annual series for consumption inequality is not available in the same way.

So the claim is not that this is a complete welfare index. The claim is that it is a structured improvement on GDP-only evaluation when the question is whether output growth actually improves worker-centered welfare.
